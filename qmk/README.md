## QMK keyboard firmware

Configurations for keyboards powered by [QMK][qmk].

* `qmk.ini` belongs in `$HOME/.config/qmk/` (Linux) / `$HOME/Library/Application Support/qmk/` (macOS)
* Keymaps under the `keyboards` directory belong in `$QMK_HOME/keyboards/`
* Keymaps depend on [getreuer's modules][gmods] to support shifted symbols of the [Engram layout][engram] (see [Community Modules][qmods])

[qmk]: https://qmk.fm/
[qmods]: https://docs.qmk.fm/features/community_modules
[gmods]: https://github.com/getreuer/qmk-modules
[engram]: https://engram.dev
